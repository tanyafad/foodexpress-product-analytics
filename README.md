# FoodExpress Product Analytics

FoodExpress: End-to-End Product Analytics Case Study

FoodExpress — мобильное приложение для заказа еды из ресторанов. Пользователь может просматривать рестораны, искать блюда, оформлять заказ, отслеживать доставку и оплачивать заказ внутри приложения.


## Data Enrichment

Исходный датасет содержал информацию только о заказах и связанных сущностях. Для моделирования задач продуктовой аналитики была разработана и сгенерирована таблица событий (event log), отражающая пользовательское поведение в приложении до оформления заказа.

Это позволило провести анализ продуктовой воронки, удержания пользователей и смоделировать A/B-тесты.
